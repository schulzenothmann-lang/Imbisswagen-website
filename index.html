01 · Einführung

Was sind Funktionsscharen?

Eine Funktionsschar ist eine Menge von Funktionen, die durch einen Parameter (oft $t$, $a$ oder $k$) beschrieben werden. Jeder Parameterwert liefert eine andere Funktion – und damit eine andere Kurve.

∑

Definition

Eine Funktionsschar $f_t$ ordnet jedem Parameterwert $t \in \mathbb{R}$ eine Funktion $f_t: \mathbb{R} \to \mathbb{R}$ zu.

$$f_t(x) = t \cdot x^2 + 1, \quad t \in \mathbb{R}$$
⟲

Parameter

Der Parameter steuert Form, Lage und Eigenschaften der Kurve. Er wirkt wie ein „Regler" für die gesamte Familie.

  • Streckt oder staucht den Graphen
  • Verschiebt Extrempunkte
  • Ändert die Anzahl der Nullstellen
◎

Ortskurve

Die Ortskurve verbindet alle Extrempunkte (oder Wendepunkte) der Schar zu einer eigenen Kurve.

Ortskurve: Menge aller Extrempunkte $E_t$ für alle $t$

Beispiel: Parabelnschar $f_t(x) = x^2 - 2tx + t$

1
Parameter einsetzen: Für $t = 0{,}5$: $f_{0,5}(x) = x^2 - x + 0{,}5$
Für $t = 2$: $f_2(x) = x^2 - 4x + 2$
2
Extrempunkt bestimmen: $f_t'(x) = 2x - 2t = 0 \Rightarrow x_E = t$
$y_E = f_t(t) = t^2 - 2t^2 + t = -t^2 + t$
3
Ortskurve: $E_t = (t,\, -t^2+t)$, also $y = -x^2 + x$
02 · Grundlagen

Wichtige Begriffe & Methoden

Diese Konzepte braucht man beim Arbeiten mit Funktionsscharen immer wieder.

Extrempunkte der Schar

Für jede Funktion $f_t$ werden Extrempunkte wie gewohnt über die Ableitung bestimmt:

  1. $f_t'(x) = 0$ lösen → gibt $x_E$ (oft abhängig von $t$)
  2. $f_t''(x_E) \neq 0$ prüfen
  3. $y_E = f_t(x_E)$ berechnen
  4. Ergebnis: $E_t = (x_E(t),\, y_E(t))$
Beispiel
$$f_t(x) = x^3 - 3tx$$ $$f_t'(x) = 3x^2 - 3t = 0$$ $$x = \pm\sqrt{t} \quad (t > 0)$$ $$y = f_t(\sqrt{t}) = -2t\sqrt{t}$$

Nullstellen der Schar

Nullstellen sind die $x$-Werte, für die $f_t(x) = 0$ gilt. Je nach Parameter kann die Anzahl variieren.

  • Löse $f_t(x) = 0$ nach $x$ auf (Parameter $t$ bleibt stehen)
  • Diskriminante zeigt: wie viele Nullstellen gibt es?
  • Für welche $t$ existieren keine / eine / zwei Nullstellen?
Beispiel
$$f_t(x) = x^2 - t$$ $$x^2 = t$$ $$x = \pm\sqrt{t} \text{ für } t > 0$$ $$\text{keine NS für } t < 0$$

Wendepunkte der Schar

Wendepunkte liegen dort, wo die Krümmung das Vorzeichen wechselt:

  1. $f_t''(x) = 0$ lösen → $x_W(t)$
  2. $f_t'''(x_W) \neq 0$ prüfen
  3. $y_W = f_t(x_W)$
  4. Ortskurve durch alle $W_t$
Beispiel
$$f_t(x) = x^4 - 6tx^2$$ $$f_t''(x) = 12x^2 - 12t = 0$$ $$x_W = \pm\sqrt{t}$$ $$y_W = 1 - 6t \cdot t = 1 - 6t^2$$

Ortskurve berechnen

Die Ortskurve entsteht, indem man den Parameter $t$ aus den Koordinaten des Sonderpunkts eliminiert:

  1. Extrempunkt $E_t = (x_E(t),\, y_E(t))$ bestimmen
  2. $t$ aus der $x$-Koordinate ausdrücken: $t = g(x)$
  3. In die $y$-Koordinate einsetzen
  4. Ergebnis: Ortskurve $y = h(x)$
Beispiel
$$E_t = (t,\; -t^2+t)$$ $$\Rightarrow x = t \Rightarrow t = x$$ $$\Rightarrow y = -x^2 + x$$ $$\text{Ortskurve: } y = -x^2 + x$$

Aufgabe 1 Extrempunkte

Gegeben ist die Funktionsschar $f_t(x) = x^2 - 2tx + 3t$, $t \in \mathbb{R}$.

  1. Bestimme die Extrempunkte $E_t$ in Abhängigkeit von $t$.
  2. Berechne die Gleichung der Ortskurve.
a) Extrempunkte:
$f_t'(x) = 2x - 2t = 0 \Rightarrow x_E = t$
$f_t''(x) = 2 > 0$ → Tiefpunkt
$y_E = t^2 - 2t^2 + 3t = -t^2 + 3t$
$$E_t = (t,\; -t^2 + 3t)$$
b) Ortskurve:
$x = t \Rightarrow t = x$ einsetzen:
$$y = -x^2 + 3x$$
Aufgabe 2 Nullstellen

Gegeben ist die Funktionsschar $f_a(x) = x^2 - a^2$, $a \in \mathbb{R}$.

  1. Berechne die Nullstellen in Abhängigkeit von $a$.
  2. Für welche Werte von $a$ hat $f_a$ keine reellen Nullstellen?
a) Nullstellen:
$x^2 = a^2 \Rightarrow x = \pm |a|$
Für $a \neq 0$: zwei Nullstellen $x_1 = a$, $x_2 = -a$
Für $a = 0$: eine Nullstelle $x = 0$
b) Da $a^2 \geq 0$ für alle $a \in \mathbb{R}$, hat $f_a$ immer mindestens eine Nullstelle. Es gibt keinen Wert von $a$, für den keine reellen Nullstellen existieren.
Aufgabe 3 Schar & Ortskurve

Gegeben ist $f_t(x) = x^3 - 3t^2 x$, $t > 0$.

  1. Bestimme alle Extrempunkte der Schar.
  2. Zeige, dass alle Hochpunkte auf der Ortskurve $y = 2x$ liegen.
a) Extrempunkte:
$f_t'(x) = 3x^2 - 3t^2 = 0 \Rightarrow x = \pm t$
$f_t''(x) = 6x$
$x = -t$: $f_t''(-t) = -6t < 0$ → Hochpunkt
$x = +t$: $f_t''(t) = 6t > 0$ → Tiefpunkt
Hochpunkt: $y_H = (-t)^3 - 3t^2(-t) = -t^3 + 3t^3 = 2t^3$
$$H_t = (-t,\; 2t^3)$$
b) Ortskurve:
$x = -t \Rightarrow t = -x$
$y = 2(-x)^3 = -2x^3$
Prüfung: $y = 2t^3$ und $x = -t$, also $t = -x$, $y = 2(-x)^3 = -2x^3$
Für $x < 0$ (da $t > 0$): $y = -2x^3 > 0$ ✓
Aufgabe 4 Gemeinsamer Punkt

Die Funktionsschar $f_t(x) = (t-1)x^2 + 2x$, $t \in \mathbb{R}$.

Bestimme alle Punkte, durch die jede Kurve der Schar verläuft (gemeinsame Punkte).

Gemeinsame Punkte erfüllen $f_t(x) = y$ für alle $t$.
$f_t(x) = (t-1)x^2 + 2x = t \cdot x^2 - x^2 + 2x$
Damit dies für alle $t$ gleich ist, muss der $t$-Anteil verschwinden:
$x^2 = 0 \Rightarrow x = 0$
$f_t(0) = 0$
$$\Rightarrow P = (0,\; 0)$$ Alle Kurven der Schar gehen durch den Ursprung.
Aufgabe 5 Schwer

Gegeben: $f_t(x) = \dfrac{x^2}{t} + t$, $t > 0$.

  1. Zeige, dass alle Extrempunkte auf der $y$-Achse liegen.
  2. Bestimme die Ortskurve der Extrempunkte.
  3. Zeige, dass $f_t(x) \geq 2|x|$ für alle $x, t > 0$ gilt.
a) $f_t'(x) = \frac{2x}{t} = 0 \Rightarrow x_E = 0$
→ Alle Extrempunkte haben $x_E = 0$, liegen also auf der $y$-Achse. ✓
b) $y_E = f_t(0) = \frac{0}{t} + t = t$
$E_t = (0,\; t)$ für $t > 0$
Ortskurve: $x = 0$, $y = t > 0$ → die positive $y$-Achse
c) Nach der AM-GM-Ungleichung gilt: $\frac{x^2}{t} + t \geq 2\sqrt{\frac{x^2}{t} \cdot t} = 2\sqrt{x^2} = 2|x|$ ✓
